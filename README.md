# Financial-Independence.github.io

Financial-Independence의 공식 포트폴리오 사이트입니다.

Financial-Independence는 웹/모바일 제품을 직접 기획, 개발, 운영하는 제품 개발 회사입니다. 이 저장소는 회사 소개, 출시 제품, 작업 방식, 문의 채널을 담은 정적 웹사이트입니다.

## 소개

이 사이트는 Financial-Independence가 만든 제품을 한곳에서 보여주기 위해 만들어졌습니다. 각 프로젝트의 목적, 주요 기능, 실제 화면, 사용 가능한 플랫폼을 소개합니다.

## 주요 페이지

- `Home`: 회사 소개와 대표 프로젝트 요약
- `Projects`: 현재 운영 중인 프로젝트 목록
- `Project`: 개별 프로젝트 상세 소개
- `About`: 작업 방식과 개발 원칙
- `Contact`: 문의, 제휴, 개발 요청 연락 창구

## 소개 중인 프로젝트

### Dividend Analysis

배당 현금흐름을 월 단위로 시뮬레이션하는 투자 분석 도구입니다. 종목 검색, 배당 재투자 비교, 월별 결과 확인 흐름을 제공하며 웹과 모바일 환경을 대상으로 합니다.

### Deposit Guard

전월세 보증금 분쟁에 대비해 입주/퇴거 상태를 기록하고 PDF 증거 자료를 생성하는 모바일 앱입니다. 체크리스트, 사진 기록, 비교 리포트 생성을 중심으로 구성되어 있습니다.
